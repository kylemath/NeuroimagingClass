BrainImation: The BCI Turing Machine (60 minutes)

What is a "BCI Turing Machine"?

A Turing Machine is a theoretical device that can compute anything computable‚Äîit's universal. BrainImation is like that for brain-computer interfaces: any BCI, neurofeedback system, or neuroart installation can be replicated in BrainImation through code. It combines real-time EEG data, creative coding (P5.js), and a live editor into one integrated platform.

What makes it universal:

  • ‚úÖ Real EEG data: Direct Bluetooth connection to Muse headsets OR realistic simulation
  • ‚úÖ Live coding environment: Write code and see results instantly‚Äîno compilation, no setup
  • ‚úÖ Full EEG access: Raw signals (256 Hz), frequency bands (alpha, beta, theta, delta, gamma), derived metrics
  • ‚úÖ Creative freedom: P5.js graphics, sound, interaction‚Äîanything you can imagine
  • ‚úÖ Research-grade: Can collect data, implement actual BCI protocols, run experiments

Key EEG Data Access Patterns

// Pattern 1: Use frequency band power (most common)
let size = map(eegData.alpha, 0, 1, 10, 100);
ellipse(width/2, height/2, size);

// Pattern 2: Use derived metrics
let brightness = eegData.attention * 100;
background(0, 0, brightness);

// Pattern 3: Access raw EEG for research
let tp9Samples = eegData.getRawChannel('TP9', 100);
for (let i = 0; i < tp9Samples.length; i++) {
  let y = map(tp9Samples[i], -100, 100, height, 0);
  point(i, y);
}

// Pattern 4: Multi-channel epoch analysis
let epoch = eegData.getRecentEpoch(128); // 0.5 seconds
// epoch[timepoint][channel] ‚Üí 4 channels √ó 128 samples
