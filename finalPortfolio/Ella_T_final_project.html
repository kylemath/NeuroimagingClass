Final Project: BrainImation Creative Project

For my final project, I built an EEG controlled Pac-Man neurofeedback game. The game

contains a Pac-Man figure with a moving mouth and a similar colour palette to the original game

design. Each orb is randomly generated, meaning there‚Äôs a new layout every time it resets. The

Pac-Man figure is controlled through beta & alpha waves (attention & meditation levels). Alpha

values control horizontal movement (left & right). Low alpha levels (0-0.4) cause movement to

the left, whereas high levels (0.6-1) cause movement to the right. Alpha levels of 0.5 cause no

horizontal movement. Beta values control vertical movement, such that low beta (0-0.5) causes

Pac-Man to move up, whereas high beta (0.5-1) makes Pac-Man move down. The speed is also

impacted by the intensity.

It acts as a neurofeedback game since movement changes with different brain states. The

higher or lower the attention/meditation controls the direction of Pac-Man, meaning you can

learn to control different mental states to collect orbs. The score system also helps track progress,

especially if paired with a stopwatch. With practice, someone could develop strategies to

manipulate their brain states enough to collect orbs more efficiently. The game could also

increase its difficulty in the future by adding ghosts that force the player to maintain even better

control. Overall it incorporates a goal-oriented task with visual feedback, making it perfect for

those who want to learn how to regulate their mental states.

Code:

let pacman;

let pellets = [];

let score = 0

function setup() {

colorMode( HSB , 360 , 100 , 100 );

pacman = {

x: width / 2 ,

y: height / 2 ,

size: 40

};

// Create pellets

for ( let i = 0 ; i < 30 ; i++) {

pellets.push({

x: random( 20 , width- 20 ),

y: random( 20 , height- 20 ),

eaten: false

});

}

}

function draw() {

background( 0 , 0 , 5 );

// --- EEG CONTROL ---

let dx = map(eegData.alpha, 0 , 1 , - 3 , 3 );

let dy = map(eegData.beta, 0 , 1 , - 3 , 3 );

pacman.x += dx;

pacman.y += dy;

// Keep inside screen

pacman.x = constrain(pacman.x, 0 , width);

pacman.y = constrain(pacman.y, 0 , height);

//Draw Pac-Man

fill( 50 , 100 , 100 );

let open = abs(sin(frameCount * 0.2 )) * 40 ; // mouth opening angle

arc(

pacman.x, pacman.y,

pacman.size, pacman.size,

radians(open),

radians( 360 - open),

PIE

---

);

// Draw pellets

fill( 200 , 80 , 100 );

for ( let p of pellets) {

if (!p.eaten) {

ellipse(p.x, p.y, 10 );

// collision detection

if (dist(pacman.x, pacman.y, p.x, p.y) < 20 ) {

p.eaten = true ;

score++;

}

}

}

// Score display

fill( 120 , 0 , 100 );

textSize( 20 );

text( "Score: " + score, 20 , 30 );

}
