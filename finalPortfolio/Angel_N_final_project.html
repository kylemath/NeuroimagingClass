Question 01

This feedback game is based on the user‚Äôs level of relaxation and focus, which are used to

control a running stickman. In the absence of real-time EEG input from the Muse headset,

simulated data was used. Higher meditation values increase the runner‚Äôs speed, while

attention levels determine jump height, enabling the runner to successfully jump over the

feedback through screen shake and background colour changes to red.

This running game relies on closed-loop neurofeedback, as the neural simulated data are

continuously measured, translated, and fed back to the user in real time. The user‚Äôs current

neurological state directly influences the environment, and the resulting visual feedback in

turn encourages adjustments in meditation and attention.
