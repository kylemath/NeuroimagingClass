Wolf Island Run: A Brain-Responsive Game

This project demonstrates a real-time brain-computer interface game that adapts to the player's mental state. The game uses EEG data to control difficulty, visual effects, and player abilities based on attention and meditation levels.

Key Features

  • Brain-Responsive Gameplay: Game difficulty automatically adjusts based on attention and meditation levels
  • Dynamic Visual Feedback: Aurora displays, sky colors, and auras respond to brain state
  • Adaptive Abilities: Player speed and jump strength increase with higher focus
  • Real-Time EEG Integration: Connects with Muse headset for live brainwave data

How to Play

  • Arrow Keys: Left/Right to move, Up to jump
  • R Key: Restart the game
  • Stay Focused: Higher attention and meditation = easier gameplay and special effects!

Technical Details

The game uses p5.js for graphics and animation, integrating with the Brainimation platform's EEG data API. Key brain metrics (alpha, beta, theta, attention, meditation) are used to dynamically adjust:

  • Island spawning patterns and gaps
  • Player movement speed and jump height
  • Visual effects including aurora displays and star fields
  • Sky gradient colors based on attention vs. meditation dominance
