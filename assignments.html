Assignment 1: EEG Data Loading and Basic Filtering

Due: Sep 22

Load a real EEG dataset in BIDS format and create visualizations comparing raw data with filtered data. This assignment introduces you to neuroimaging data structures and basic signal processing.

Skills Focus: BIDS format, data loading, basic filtering, scientific plotting
Dataset: OpenNeuro ds003061 - EEG during Visual Working Memory
Specific File: sub-01/eeg/sub-01_task-workingmemory_eeg.set

Create a Python or MATLAB script that:
  • Downloads and loads the specified EEG file using MNE-Python or EEGLAB
  • Plots 10 seconds of raw EEG data from all channels
  • Applies a bandpass filter (1-40 Hz) to the same data segment
  • Creates a side-by-side comparison plot (raw vs filtered)
  • Calculates and plots the power spectral density for both conditions
  • Includes proper axis labels, titles, and a brief code comment explaining each step
Submission: Submit your script file (.py or .m) and the generated plots as a single PDF
